# Woo Sales Booster

A WooCommerce plugin with 3 powerful features in one:

- 📱 **WhatsApp Order Button** — Let customers order via WhatsApp
- 💰 **COD Extra Fee** — Charge extra on Cash on Delivery
- 🎁 **Buy X Get Y Discount** — Smart cart discount rules

## Status
🚧 In Development — Phase 1 (WhatsApp Button)

## Requirements
- WordPress 6.0+
- WooCommerce 7.0+
- PHP 7.4+
